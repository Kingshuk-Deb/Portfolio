A) Introduction - top

Ratsaan is a Clean Look, professional HTML5 Portfolio template that is specially designed for agency and personal portfolio websites. It is very content rich and tells the visitors everything they need to know about your company. This HTML5 Template also features Twitter Bootstrap3 Framework, CD word Slider, Flex Slider to ensure a better template. We can assure that this template will be a reliable, trustworthy business template for you.

Template Features

  • Latest Bootstrap 4.3.1
  • Multipage and landing template
  • Sticky Navigation
  • HTML5 Validated
  • Responsive Template
  • FontAwesome icons
  • Themeify Icons
  • Google Web Fonts
  • Stylist blogSingle Page
  • Working Contact form
  • Documentation
  • Easy to customize and user friendly
